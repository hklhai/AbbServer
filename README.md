## Maven使用

## Tip
建议使用IDEA开发，先建立task，然后创建branch，在branch下开发；测试无误后，切换至master后与branch比对，再次确认无误后merge，最后commit And Pull至master即可。
若不采用IDEA提交，可使用第三方SourceTree，SourceTree使用教程联系小豆苗。
